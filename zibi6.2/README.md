适用于最新版6.2版本

首先搭建一个站点，绑定api.zibll.com域名，并开启SSL（随便找个域名的证书就行）

上传以下源码
vkceyugu.cdn.bspapp.com/VKCEYUGU-cc8cf08f-49f5-4fc5-83c3-ed2a683704d4/0122eb75-5328-4aa0-a9bb-e1f2d6a79ee9.zip
里面包含伪静态规则

最后在/etc/hosts里面加一行
127.0.0.1 api.zibll.com
